## Описание проекта

WPF-приложение «Калькулятор чаевых», которое запрашивает:
- Сумму счёта
- Количество гостей (для разделения счёта)
- Процент чаевых (без чаевых / 5% / 10% / 15%)

После нажатия кнопки «Рассчитать» выводит:
- Итоговую сумму с чаевыми
- Сумму на каждого гостя (если гостей больше одного)
